User Guide

Learn how to navigate and utilize the RED Rams platform effectively.

Getting Started

Welcome to Two Steep Ahead, your go-to platform for forecasting landslide risks across the Philippines. With this web app, you can easily access a range of tools designed to help you analyze and evaluate possible landslide occurence on a given location. Start exploring the available features and gain valuable insights to make informed decisions regarding landslide occurence in your area.

Using the Dynamic Map

The Dynamic Map is a tool for landslide forecasting specifically within the Philippines. Users select a location by searching or clicking the map. Upon selection, the system automatically retrieves fixed location data (coordinates, soil type, slope). Users then select a date up to 5 days ahead to fetch dynamic weather data (rainfall) from Open-Meteo. Combining the fixed location data with the weather forecast, clicking "Predict" analyzes the variables to determine the likelihood of a landslide at that location on the specified date, providing a prediction result and a confidence score.

Understanding the Dashboard

The dashboard provides a visual overview of landslide data, historical trends, and predictive analytics. Here, you can track vulnerable areas, view forecasts, and access detailed reports on landslide incidents in various regions.

Viewing Training Results

The training results section gives insights into how machine learning models predict landslide occurrences. You can analyze past data, view model accuracy, and see predictive results on future events. This section helps in understanding the power of prediction analysis in risk management.
